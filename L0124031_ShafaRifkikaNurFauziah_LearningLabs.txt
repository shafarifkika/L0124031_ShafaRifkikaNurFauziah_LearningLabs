Disini saya membuat web yang bertema yaitu tentang kisah saya.

Jadi dari web ini saya menggunakan html, css, dan java script.
dari html saya membuat 
1. index.html
   indext.html berisikan tampilan Utama web saya setelah mengklik link apabila sudah di upload. Saya menggunakan ul li untuk  mengisi apa saja yang ingin saya masukan dalam web. Sesuai dengan ketentuan yang telah ditetapkan yaitu membutuhkan index.html,content.html,feedback.htnl, css,  javascript, dan image.

4. Image
